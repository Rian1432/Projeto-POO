### Tema do Projeto:
Sistema de venda de computadores

### Regras de negócio:
Cada computador no estoque vai ser catalogado com um tipo de uso (básico/performance/jogos), e por sua portabilidade(portátil/não portátil). Também será cadastrado cada venda de computador feita no sistema juntamente com o cliente para qual foi vendido, e o vendedor que efetuou a venda.
